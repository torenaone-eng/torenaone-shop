# TorenaOne — интернет-магазин тепловых насосов

Прототип одно-брендового интернет-магазина TorenaOne (тепловые насосы для дома, бассейна, ГВС и накопительные ёмкости). Дизайн в фирменном эко-стиле, рынок РФ/СНГ.

## Что внутри

- `index.html` — кликабельный прототип магазина (одностраничное приложение, без сборки). Содержит: главную, каталог с фильтрами (категория, хладагент, тип, мощность, цена), карточку товара, калькулятор подбора оборудования, корзину и оформление заказа с выбором оплаты СБП / при получении.
- `images/` — фото товаров по сериям (вырезаны из заводского каталога).
- `data/catalog-torenaone.json` — каталог из 89 моделей (источник данных): категории, серии, характеристики, цены, описания, особенности.
- `docs/` — структура сайта и документ архитектуры.

## Как открыть

Открыть `index.html` в браузере. Внешних зависимостей и сборки нет — данные каталога встроены в страницу, фото подгружаются из `images/`.

## Каталог

89 моделей в 4 категориях: для дома (32), для бассейна (37), горячая вода (13), ёмкости (7).
Оборудование с хладагентом R410a исключено. Серия Wine Glass переименована в Style.

## Статус

Это прототип для согласования. Дальнейшие шаги: перенос полных тех. характеристик по моделям, интеграции (МойСклад, CRM, платёжный сервис СБП), вёрстка боевого фронтенда и бэкенда.
